## 知识库内容

### 1. 项目基本信息

- **项目名称**: 需求挖掘系统
- **项目类型**: 全栈 Web 应用
- **技术栈**: React + FastAPI + SQLite + AI/ML
- **核心功能**: 基于语义聚类的需求挖掘

### 2. 三大模块

1. **词根聚类模块** - 从英文单词种子发现产品机会
2. **商品管理模块** - 分析市场供给情况
3. **Reddit 模块** - 从讨论中挖掘需求（预留）

### 3. 核心文档

- `docs/需求文档.md` - 统一需求文档
- `docs/使用手册.md` - 完整使用说明
- `docs/design/architecture.md` - 架构设计
- `docs/design/database-design.md` - 数据库设计
- `docs/design/api-design.md` - API 设计
- `.claude/project-context.md` - 项目上下文

### 4. 核心原则

- **只分组，不删除** - 系统提供视图，人工做决策
- **轻量版起步** - 避免复杂度压垮
- **大模型输出是假设** - 需要验证
