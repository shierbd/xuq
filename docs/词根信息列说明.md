# Phase 0 词根信息列详细说明

## 概述

Phase 0 分词结果中现在包含了**完整的词根追踪信息**，帮助您全面了解每个词汇的来源和用途。

## 词根信息列详解

### 1. 是否为词根 (Is Seed Word)

**含义**: 该词汇是否被用作原始词根（seed_word）来扩展关键词

**取值**:
- `是` - 该词在数据库中作为seed_word使用，有关键词从它扩展而来
- `否` - 该词仅作为分词词汇出现，未被用作扩展词根

**用途**:
- 快速识别哪些词是您的原始词根
- 筛选时可排除已知词根，专注于发现新词

**示例**:
```
| 词汇       | 是否为词根 | 说明                           |
|-----------|-----------|--------------------------------|
| calculator| 是        | 作为词根扩展了805个关键词       |
| scientific| 是        | 作为词根扩展了213个关键词       |
| online    | 否        | 只是分词词汇，未用作词根        |
| best      | 是        | 作为词根扩展了1,368个关键词     |
```

---

### 2. 作为词根的扩展数 (Expansion Count as Seed)

**含义**: 如果该词是词根，它扩展出了多少个关键词短语

**取值**:
- `0` - 不是词根
- `> 0` - 该词作为seed_word扩展的phrase数量

**用途**:
- 评估词根的扩展效果
- 发现高价值词根（扩展数适中）
- 识别过度宽泛的词根（扩展数过多）

**参考标准**:
- `0` - 普通分词词汇
- `1-50` - 小众词根，可能需要更多扩展
- `50-300` - **高价值词根**，适合深入挖掘
- `300-1000` - 热门词根，已有大量相关词
- `>1000` - 超热门词根，可能过于宽泛

**示例**:
```
| 词汇       | 作为词根的扩展数 | 评估                    |
|-----------|----------------|------------------------|
| code      | 1,749          | 超热门，词义宽泛        |
| best      | 1,368          | 超热门，通用修饰词      |
| calculator| 805            | 热门专业词，价值高      |
| scientific| 213            | 高价值词根，值得深挖    |
| graphing  | 42             | 小众专业词，可继续扩展  |
| online    | 0              | 非词根，分词词汇        |
```

---

### 3. 来源词根 (Source Seeds)

**含义**: 该词汇出现在哪些seed_word的短语中（即该词是从哪些词根的关键词中分词而来）

**格式**:
- 多个词根用逗号分隔
- 超过5个时显示前5个，后面加 `... (+N个)`

**取值示例**:
- `calculator` - 只在calculator词根的短语中出现
- `best, calculator, phone, laptop, chart` - 在5个不同词根的短语中出现
- `free, best, top, cheap, online... (+15个)` - 在20个词根的短语中出现

**用途**:
- 了解词汇的应用场景广度
- 判断词汇的通用性
- 来源词根少 = 专业性强，适合扩词
- 来源词根多 = 通用词汇，扩词价值低

**示例**:
```
| 词汇          | 来源词根                         | 来源词根数 |
|--------------|----------------------------------|-----------|
| graphing     | calculator                       | 1         |
| scientific   | calculator                       | 1         |
| tattoo       | tattoo                           | 1         |
| design       | tattoo, chart, calculator        | 3         |
| best         | calculator, tattoo, code... (+86)| 89        |
| free         | calculator, tattoo, code... (+86)| 89        |
```

---

### 4. 来源词根数 (Source Seed Count)

**含义**: 该词汇来源于多少个不同的seed_word

**取值**: 整数，表示唯一词根数量

**用途**:
- **快速判断词汇专业性**
- 来源数 = 1-3: 专业词汇，高价值
- 来源数 = 4-10: 较专业，有一定价值
- 来源数 > 10: 通用词汇，扩词价值低

**筛选建议**:
```
来源词根数 <= 5: 筛选专业性强的词汇
来源词根数 >= 20: 识别需要添加到停用词的通用词
```

**示例**:
```
| 词汇          | 来源词根数 | 评估                        |
|--------------|-----------|----------------------------|
| graphing     | 1         | 高度专业，calculator领域    |
| vector       | 2         | 较专业，calculator+chart    |
| design       | 3         | 有一定专业性                |
| tool         | 12        | 较通用                      |
| best         | 89        | 极度通用，建议加停用词      |
| free         | 89        | 极度通用，建议加停用词      |
```

## 四列配合使用指南

### 场景1: 发现高价值扩词候选

**筛选条件**:
- `是否为词根` = 否 （不是已知词根）
- `作为词根的扩展数` = 0 （未被用作词根）
- `来源词根数` <= 5 （专业性强）
- `频次` >= 10 （出现频繁）

**结果**: 高频专业词汇，适合用作下一轮扩词的seed_word

---

### 场景2: 评估现有词根效果

**筛选条件**:
- `是否为词根` = 是
- 按 `作为词根的扩展数` 排序

**分析**:
- 扩展数 50-300: 最优词根，值得继续深挖
- 扩展数 < 50: 可能需要更多扩展
- 扩展数 > 1000: 可能过于宽泛，考虑细分

---

### 场景3: 识别需要添加的停用词

**筛选条件**:
- `来源词根数` >= 20 （出现在太多词根的短语中）
- `是否为词根` = 否 （不是词根，只是通用词）

**结果**: 通用修饰词（如best, top, free, cheap），建议加入停用词列表

---

### 场景4: 追踪词汇扩展路径

**操作**:
1. 查看 `来源词根` 列，了解词汇来源
2. 如果来源单一，说明从特定词根扩展而来
3. 如果来源众多，说明是跨领域通用词

**示例追踪**:
```
第1轮: 导入seed_word = "calculator"
  ↓
分词得到: scientific, graphing, online, tool

第2轮: 选择"scientific"作为新seed_word扩展
  ↓
分词得到: research, method, calculator...

查看"calculator"的来源词根:
- 第1轮: 来源 = [calculator] （原始词根）
- 第2轮: 来源 = [calculator, scientific] （同时来自两个词根）
```

## 常见问题

### Q1: 为什么有些词既是词根，又出现在其他词根的短语中？

**A**: 这是正常的迭代扩词结果。

**示例**:
```
第1轮: seed_word = "calculator"
  → 扩展出 "scientific calculator"
  → 分词得到 "scientific"

第2轮: seed_word = "scientific" (用第1轮的分词词作为新词根)
  → 扩展出 "scientific method"
  → 此时"scientific"既是词根(扩展数>0)，也来源于calculator
```

### Q2: 来源词根数=89是不是太多了？

**A**: 是的！这说明该词极度通用。

**原因**: 您的数据库中有89个不同的seed_word，像"best"、"free"这样的通用词会出现在几乎所有词根的短语中。

**建议**: 将来源词根数>20的通用词添加到停用词列表。

### Q3: "作为词根的扩展数"为0，但"是否为词根"为"是"？

**A**: 不会出现这种情况。

**逻辑**:
- `是否为词根 = 是` ⟺ `作为词根的扩展数 > 0`
- `是否为词根 = 否` ⟺ `作为词根的扩展数 = 0`

### Q4: 如何理解"来源词根"和"作为词根的扩展数"的区别？

**A**:
- **来源词根**: 该词是从哪些词根的短语中分词而来（被动）
- **作为词根的扩展数**: 该词作为词根主动扩展了多少个短语（主动）

**类比**:
- 来源词根 = 我的父母是谁（我的来源）
- 扩展数 = 我有几个孩子（我作为父母的扩展）

**示例**:
```
"scientific"词的信息：
- 来源词根: [calculator] （从calculator的短语中分词而来）
- 作为词根的扩展数: 213 （"scientific"作为词根扩展了213个短语）
```

## 总结

通过这四列词根信息，您可以：

1. ✅ **识别高价值扩词候选** - 专业性强、频次高、未被用作词根的词
2. ✅ **评估词根效果** - 通过扩展数判断词根是否需要继续深挖
3. ✅ **清理停用词** - 识别通用修饰词并添加到停用词列表
4. ✅ **追踪扩词路径** - 了解每个词的来源和用途
5. ✅ **优化扩词策略** - 基于数据选择下一轮扩词方向

**关键指标速查**:
- 来源词根数 ≤ 5 → 专业词汇 ⭐⭐⭐
- 作为词根的扩展数 50-300 → 高价值词根 ⭐⭐⭐
- 来源词根数 ≥ 20 → 通用词，考虑停用 ⚠️
- 作为词根的扩展数 > 1000 → 过于宽泛 ⚠️
