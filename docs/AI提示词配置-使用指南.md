## 🎯 功能概述

现在您可以为每个AI使用场景自定义提示词模板了！提示词是发送给AI的指令，用于指导AI如何处理任务。

## 📍 访问地址

**前端页面**: http://localhost:3000/ai-config

在左侧菜单点击：**AI配置管理 → AI配置**

## 🚀 如何配置提示词

### 第1步：配置使用场景

1. 访问 AI配置页面
2. 展开"高级配置（可选）"部分
3. 点击"一键配置使用场景（推荐）"按钮
4. 系统会自动创建3个预设场景：
   - 类别名称生成
   - 需求分析
   - 交付产品识别

### 第2步：配置提示词

1. 在"已配置的场景"列表中，找到您想配置的场景
2. 点击场景右侧的"配置提示词"按钮
3. 在弹出的对话框中：
   - **提示词名称**：给提示词起个名字（例如：类别名称生成提示词）
   - **提示词模板**：输入您的提示词内容

### 第3步：保存提示词

1. 填写完提示词后，点击"保存提示词"按钮
2. 系统会保存您的提示词并自动激活
3. 下次该场景调用AI时，会使用您配置的提示词

## 📝 提示词模板示例

### 示例1：类别名称生成

```
你是一个专业的产品分类专家。请根据以下关键词生成一个简洁的类别名称（2-4个单词）。

关键词：{keywords}

要求：
1. 名称要简洁明了
2. 能准确概括关键词的共同特征
3. 使用英文
4. 只返回类别名称，不要其他内容

示例：
- 关键词：laptop, computer, notebook → "Laptop Computers"
- 关键词：phone, mobile, smartphone → "Mobile Phones"
```

{products}

请按以下格式输出：

**用户需求**：
- [需求1]
- [需求2]
- [需求3]

**目标用户**：
- [用户群体1]
- [用户群体2]

**使用场景**：
- [场景1]
- [场景2]

要求：
1. 分析要深入、具体
2. 需求要真实、可验证
3. 用户群体要明确
4. 场景要具体、可操作
```

### 示例3：交付产品识别

标题：{title}

请按以下格式输出：

**交付类型**：[实体商品/数字商品/服务]

**交付格式**：[具体格式，如PDF、视频、实物等]

**交付平台**：[平台名称，如网站、邮件、快递等]

**说明**：[简要说明交付方式]

要求：
1. 准确识别交付类型
2. 明确交付格式
3. 说明要简洁清晰
```

## 💡 提示词编写技巧

### 1. 明确角色定位

在提示词开头明确AI的角色：
```
你是一个专业的[角色名称]。
```

例如：
- "你是一个专业的产品分类专家。"
- "你是一个经验丰富的需求分析师。"
- "你是一个资深的市场研究员。"

### 2. 清晰的任务描述

明确告诉AI要做什么：
```
请根据以下[输入]，[执行的任务]。
```

例如：
- "请根据以下关键词，生成一个类别名称。"
- "请分析以下商品，识别用户需求。"

### 3. 使用变量

使用 `{变量名}` 来标记动态内容：
```
关键词：{keywords}
商品列表：{products}
标题：{title}
描述：{description}
```

### 4. 明确输出格式

告诉AI如何组织输出：
```
请按以下格式输出：

**标题1**：
- 内容1
- 内容2

**标题2**：
- 内容3
```

### 5. 添加要求和约束

列出具体要求：
```
要求：
1. [要求1]
2. [要求2]
3. [要求3]
```

### 6. 提供示例（可选）

给出输入输出示例：
```
示例：
- 输入：[示例输入]
- 输出：[示例输出]
```

```

```

```
关键词：laptop, computer, notebook
```

## ❓ 常见问题

### Q1: 提示词可以多长？

建议提示词长度在 100-1000 字之间。太短可能不够清晰，太长可能影响效率。

### Q2: 可以使用中文吗？

可以！提示词可以使用中文、英文或混合使用。

### Q3: 如何测试提示词效果？

1. 配置提示词后，使用相应的功能
2. 查看AI输出结果
3. 根据效果调整提示词
4. 重新保存并测试

### Q4: 可以为同一个场景配置多个提示词吗？

可以！系统支持版本管理，但同一时间只有一个提示词是激活状态。

### Q5: 如何修改已有的提示词？

1. 点击场景的"配置提示词"按钮
2. 系统会加载当前激活的提示词
3. 修改内容后点击"保存提示词"
4. 系统会更新提示词

### Q6: 提示词会影响所有功能吗？

不会。提示词只影响对应的使用场景。例如，"类别名称生成"的提示词只影响类别名称生成功能。
