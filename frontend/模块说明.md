# 前端模块说明

## 📊 模块概览

需求挖掘系统 v2.0 前端包含 **2个独立模块**：

### 1️⃣ 词根聚类模块 (Keywords Module)
**目录**: `frontend/src/pages/keywords/`
**页面数**: 8个
**路由前缀**: `/seed-words`, `/keyword-import`, `/keywords`, `/clusters`, `/cluster-annotation`, `/direction-selection`, `/directions`

### 2️⃣ 商品管理模块 (Products Module)
**目录**: `frontend/src/pages/products/`
**页面数**: 4个
**路由前缀**: `/`, `/import`, `/batch-import`, `/export`

---

## 🎯 词根聚类模块详细说明

### 模块目标
从英文单词种子发现产品机会方向

### 工作流程
```
A1 种子词准备 → A2 短语扩展 → A3 语义聚类 → A4 AI簇标注 → A5 方向筛选
                                                                    ↓
                                                            B阶段：方向深化
                                                            (B1→B3→B6)
```

### 页面列表

#### A阶段页面（种子词扩展与聚类）

| 页面 | 路由 | 文件 | 功能说明 |
|------|------|------|----------|
| A1-种子词管理 | `/seed-words` | `SeedWordManagement.jsx` | 管理46个英文单词种子，支持添加、编辑、删除、批量导入导出 |
| A2-关键词导入 | `/keyword-import` | `KeywordImport.jsx` | 导入SEMrush/Reddit采集的关键词数据（CSV/Excel），支持拖拽上传 |
| 关键词列表 | `/keywords` | `KeywordList.jsx` | 浏览和搜索6,565个关键词，支持按阶段、簇ID筛选 |
| A3-簇概览 | `/clusters` | `ClusterOverview.jsx` | 展示63个语义簇的概览，显示簇大小、种子词、搜索量等 |
| 簇详情 | `/clusters/:id` | `ClusterDetail.jsx` | 查看单个簇的详细信息，包含簇内所有关键词 |
| A4-AI簇标注 | `/cluster-annotation` | `ClusterAnnotation.jsx` | 使用AI为簇生成标签和解释，支持批量标注和自定义提示词 |
| A5-方向筛选 | `/direction-selection` | `DirectionSelection.jsx` | 从已标注簇中筛选5-10个有价值方向，评估优先级和商业价值 |

#### B阶段页面（方向深化与需求分析）

| 页面 | 路由 | 文件 | 功能说明 |
|------|------|------|----------|
| B阶段-方向管理 | `/directions` | `DirectionManagement.jsx` | 管理方向深化流程，包含B1扩展、B3聚类、B6需求分析（5维框架） |

### 数据流转

```
种子词 (46个)
    ↓ A2扩展
关键词 (6,565个)
    ↓ A3聚类
语义簇 (63个)
    ↓ A4标注
已标注簇
    ↓ A5筛选
方向 (5-10个)
    ↓ B1扩展
方向关键词 (500-2000/方向)
    ↓ B3聚类
子簇 (10-30/方向)
    ↓ B6分析
需求报告 (5维框架)
```

## 🛍️ 商品管理模块详细说明

### 模块目标
分析市场供给情况，识别交付产品类型

### 工作流程
```
P1.1 数据导入 → P1.2 数据管理 → P2.1 语义聚类 → P2.2 聚类展示 → P3 AI增强 → P1.3 数据导出
```

### 页面列表

| 页面 | 路由 | 文件 | 功能说明 |
|------|------|------|----------|
| P1-商品管理 | `/` | `ProductManagement.jsx` | 商品列表、筛选与聚类，支持编辑/删除与需求分析 |
| P1-数据导入 | `/import` | `DataImport.jsx` | 导入Etsy商品数据（Excel/CSV），支持拖拽上传 |
| P1-批量导入 | `/batch-import` | `BatchImport.jsx` | 批量导入文件夹内的商品数据文件 |
| P1-数据导出 | `/export` | `DataExport.jsx` | 导出各阶段数据（原始、聚类、汇总） |

### 数据流转

```
Etsy商品数据
    ↓ P1.1导入
商品数据库 (345个)
    ↓ P2.1聚类
商品簇 (2个有效簇)
    ↓ P3分析
需求分析与交付识别
    ↓ P1.3导出
分析报告
```

## 🔗 模块间关系

### 独立性
- 两个模块**完全独立**，可以单独运行
- 各自有独立的数据库表
- 各自有独立的API端点
- 各自有独立的页面目录

### 协同性
- 词根聚类模块发现**需求方向**
- 商品管理模块分析**供给情况**
- 两者结合实现**需求-供给匹配分析**

### 数据关联（可选）
```
词根聚类模块的"方向关键词"
        ↓
可以用于搜索商品管理模块的商品
        ↓
分析该方向的市场供给情况
```
